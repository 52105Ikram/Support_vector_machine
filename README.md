# Support Vector Machine (SVM) Machine Learning Project

## 📌 Project Overview

This project demonstrates the implementation of a **Support Vector Machine (SVM)** machine learning algorithm using Python and Scikit-learn.

The main purpose of this project is to understand how the SVM algorithm can be used for **classification problems**. The project includes data preprocessing, model training, prediction, and model evaluation.

## 🎯 Objectives

* Understand the Support Vector Machine algorithm.
* Load and explore a dataset using Pandas.
* Prepare data for machine learning.
* Split the dataset into training and testing sets.
* Train an SVM classification model.
* Make predictions on test data.
* Evaluate model performance using accuracy and other metrics.
* Visualize the model results where applicable.

## 🤖 Machine Learning Algorithm

### Support Vector Machine (SVM)

Support Vector Machine is a supervised machine learning algorithm commonly used for classification and regression tasks.

In classification, SVM finds an optimal decision boundary, called a **hyperplane**, that separates different classes of data. The algorithm tries to maximize the margin between the classes.

In this project, SVM is used to classify data into different categories based on the selected features.

## 📊 Model Evaluation

The trained model can be evaluated using:

* Accuracy Score
* Classification Report
* Confusion Matrix

These evaluation techniques help measure how well the SVM model performs on unseen test data.
